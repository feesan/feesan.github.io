# Neovim 的配置
## 插件加载器
关于插件加载器使用`LazyVim`插件来配置  
便携的安装方式 简易的使用方式 都是极大的加分项  
我不可否认地认为其是对于`新手`最简单的选择
## 关于复刻IDE功能
下列是几个基本功能
- 代码补全与LSP(Language Server protocol)语法检查  
- 括号补全功能
- 主题的使用(这可能是最重要的一点)
- 文件树 缓冲区 git提醒 状态栏 搜索栏
